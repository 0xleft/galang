## Install

### Windows

Download installer from [here](https://github.com/0xleft/gal/releases/latest/download/gal_installer.exe)

```powershell
gal.exe
```

### Linux

```bash
curl https://raw.githubusercontent.com/0xleft/gal/main/install.sh | sudo bash
gal
```
